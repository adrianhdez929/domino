# Domino

> Proyecto de Programación II.
> Facultad de Matemática y Computación - Universidad de La Habana.
> Curso 2022.

## Requisitos
- .NET 6.0

## Uso
- Compilar los sources
    - Linux y MacOS
    > ```make build```
    - Windows
    > ```dotnet build```
- Correr la app visual
    - Linux y MacOS
    > ```make run```
    - WIndows
    > ```dotnet run --project UI```
